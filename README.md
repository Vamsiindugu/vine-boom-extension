# Vine Boom HQ Longer

Plays customizable meme sound effects on syntax or compilation errors in Visual Studio Code.

![Icon](icon.png)

## Features

- **Instant Feedback**: Hear a "Vine Boom" (or other meme sounds) the moment you create a syntax error.
- **11 Iconic Sounds**: Choose from a variety of sounds including:
  - Vine Boom
  - Galaxy meme
  - Brother eww
  - Spiderman meme song
  - and more!
- **Live Preview**: Preview sounds directly in the command palette as you scroll.
- **Cross-Platform**: Works on Windows, macOS, and Linux.

## Usage

1. **Trigger**: Simply type some invalid code to create a red-squiggle syntax error.
2. **Change Sound**:
   - Open the Command Palette (`Ctrl+Shift+P` or `Cmd+Shift+P`).
   - Type **"Error Sounds: Change Sound"**.
   - Use arrow keys to preview and `Enter` to select.

## Requirements

- For **Windows**: Requires Windows Media Player to be enabled (default on most systems).
- For **Linux**: Requires `paplay`, `mpg123`, or `ffplay` to be installed.

## Extension Settings

This extension contributes the following settings:

* `errorSounds.selectedSound`: The selected meme sound to play on errors.
